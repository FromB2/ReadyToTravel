여행지 선택

여행지둘러보셈 마음에드는 여행지를 눌러서 일정만들기 버튼을 누르셈

날짜 선택

계획생성 페이지 날짜를 먼저 선택하셈

경로 선택

출발지 도착지 경유지 순으로 마커를 찍으셈 ( 경로최적화 ) 확인하셈

일차별 저장

여행 상세설정 버튼을 누르고 방금찍은마커가 몇일차 계획인지 등록하셈

계획정보 입력

선택한 기간(날짜)동안 여행할 계획의 이름 과 내용, 여행종류를 적으셈(혼자,가족,커플)

계획 만들기
